# OptiCrop 🌿 OptiCrop is a Flask-based machine learning web application that recommends a suitable crop using: - Nitrogen - Phosphorous - Potassium - Temperature - Humidity - Soil pH - Rainfall ## Features - Responsive agricultural-themed user interface - Home, About and Find Your Crop pages - Machine learning prediction using Logistic Regression - Input validation - Prediction confidence display - Ready to upload to GitHub ## Project Structure ```text OptiCrop-GitHub/ ├── app.py ├── train_model.py ├── model.pkl ├── requirements.txt ├── Procfile ├── data/ │ └── crop_recommendation_sample.csv ├── templates/ │ ├── base.html │ ├── index.html │ ├── about.html │ └── predict.html └── static/ ├── css/style.css └── js/main.js ``` ## Run Locally ```bash python -m venv venv ``` Windows: ```bash venv\Scripts\activate ``` Install dependencies: ```bash pip install -r requirements.txt ``` Start the application: ```bash python app.py ``` Open: ```text http://127.0.0.1:5000 ``` ## Train the Model Again ```bash python train_model.py ``` To use the original Kaggle dataset, place it inside the `data` folder and update `DATA_PATH` in `train_model.py`. ## Upload to GitHub ```bash git init git add . git commit -m "Initial OptiCrop application" git branch -M main git remote add origin YOUR_GITHUB_REPOSITORY_URL git push -u origin main ``` ## Important Note The included CSV is a generated demonstration dataset so the project runs immediately. For academic evaluation, replace it with the official Crop Recommendation dataset and retrain the model.
